# IAP_916

## How to use the code?
1. Open ingWizard;
2. Import two Project;
3. Use BOOT_916 EXAMP to download platform, BOOT, APP code to ing916xx chip;
4. Use USB tool to update APP code. The platform and BOOT do not support updates.
